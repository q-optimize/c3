c3po.control package¶

Submodules¶

c3po.control.goat module¶

Methods to create the objects needed for GOAT propagation

c3po.control.goat.get_initial_state(u_init, n_params)¶

Extend initial unitary by one per parameter

c3po.control.goat.get_step_matrix(h_sys, grad_list)¶

The GOAT Hamiltonian that contains the physical Hamiltonian and its gradients. It has the form: [ H , 0, 0 … 0] [dH_dp1, H, 0 … 0] [dH_dp2, 0, H … 0] [ … H]

c3po.control.goat.select_derivative(u, n_params, pos)¶

Unwrap derivatives from the propagated GOAT-vector. Position 0 returns the regular time evolution, the k-th position return gradient to the k-th parameter.
